MiniVue — build Vue 3 from scratch

Live demos for every layer. Each page imports our source directly, with no build step.

  • Layer 1 — Reactivity (ref, reactive, computed, effect)
  • Layer 2 — Virtual DOM + renderer (h, diff, keyed lists)
  • Layer 3 — Components (setup, props, emit, slots, createApp)
  • Layer 4 — Template compiler (template, {{}}, v-if, v-for, :bind, @on)
  • Layer 5 — Router (RouterView, RouterLink, :params, guards)
  • Layer 6 — Store (createPinia, defineStore, getters, actions)
  • Layer 7 — SSR (renderToString + hydrate; node playground/07-ssr/server.js)
  • Layer 8 — Forms and bindings (v-model, :class/:style, modifiers)
  • Layer 10 — Directives and dynamics (v-focus, component :is, v-model on components)
  • Layer 11 — Built-in components (Teleport, KeepAlive, async)
  • Capstone — the “Notes” app (router + store + forms + Teleport)
